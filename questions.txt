clone 的时候会遇到的问题： OpenSSL SSL_read: SSL_ERROR_SYSCALL, errno 10054

解决办法：
1、打开 git-bash 终端
2、git config http.sslVerify "false"

	提示：fatal: not in a git directory

3、git config  --global   http.sslVerify "false"
